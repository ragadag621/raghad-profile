Project:

Face Recognition System

Description:

Developed a full-stack face recognition web application capable of identifying individuals from images or live video streams. The system includes real-time face detection, feature extraction, and identity classification. I implemented the backend using Flask and integrated it with a responsive frontend using HTML/CSS and JavaScript. Through this project, I learned to manage complete project architecture, optimize model inference, and deliver functional user interfaces.

Project Goals:

Build an intelligent system that can recognize and distinguish between human faces in real time. Ensure the system can work both with static images and live video feeds using a webcam. Develop a complete full-stack web interface for user interaction.

How It Works:

1. Face Detection: Using OpenCV’s Haar cascades or DNN models to detect faces in the frame.
2. Feature Extraction: Each detected face is processed to extract unique features (face embeddings) using deep learning models.
3. Recognition/Classification: The system compares the extracted features against a known database using distance metrics.
4. Web Interface: A frontend in HTML/CSS/JS interacts with a Flask backend handling all ML logic.

Key Features:

- Real-time webcam face recognition.
- Upload and store new faces.
- High accuracy through feature-based comparison.
- Integrated ML with interactive web UI.
